# Online Learning System

A frontend-only web application for online course management with professional, academic-friendly design.

## Features

- **User Management**: Student and Instructor registration/login (mock authentication)
- **Course Catalog**: Browse available courses with detailed information
- **Enrollment System**: Enroll in courses with simulation
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Mock Data**: Uses localStorage and JSON for data simulation

## Technology Stack

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- LocalStorage for data persistence

## Pages

1. **Home Page** - System overview and features
2. **Login/Register** - User authentication (mock)
3. **Course Listing** - Display all available courses
4. **Course Details** - Detailed course information
5. **Enrollment** - Course enrollment process

## Local Development

1. Open `index.html` in a web browser
2. No server setup required - runs as static files

## Azure Deployment

This application is ready for Azure App Service deployment:

1. Zip all files in the project folder
2. Deploy to Azure App Service (Free Tier)
3. The `web.config` file ensures proper routing
